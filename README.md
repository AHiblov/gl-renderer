# OpenGL-based Renderer

## Graphical Features
1. Deferred shading
2. Forward shading
3. Waves
4. Texture Bombing
5. Triplanar mapping
6. Still particles (grass)
7. Normalmap
8. Specular
9. Fog/mist
10. Skybox

## Postprocessing Features
1. Drops on lens
2. Vignette
3. Emboss
4. Grayscale

## Graphics APIs
 - OpenGL 4.5 (Direct State Access)
 - OpenGL 3.3

## Main Application
1. Multithreading: day and night simulation
2. Hide far objects and chunks
3. Camera controllers: free-fly and first-person cameras
4. Transparent textures (available as forward shading for both shading types)
5. GUI (ImGUI library)

Examples of some features can be seen in `gallery` folder.

## Operating Systems
- Windows
- Linux
